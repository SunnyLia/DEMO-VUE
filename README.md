###  项目运行

  全局安装 vue-cli  
  $ npm install -g vue-cli  
  创建一个基于 "webpack" 模板的新项目  
  $ vue init webpack my-project  
  安装依赖，走你  
  $ cd my-project  
  $ npm install  
  $ npm run dev
